# Changelog

## [1.1.0] - 2026-03-30

- Added palm tree interaction; you can now grow Coconuts from them.
- Added willow interaction; you can now grow Kingberries from them.
- Added dead tree interaction; you can now grow Winterberries from them.
- Added cactus interaction; you can now grow Prickleberries from them.
- Added waterfall interaction; you can now refill Watering Cans under them.
- Added clean shallow water interaction in Roots; you can now refill Watering Cans with them.
- Added campfire interaction; you can now put them out.
- Added cooking interaction; water now vaporizes when cooking a Watering Can.
- Added visual and audio feedback when growing fruits with a Watering Can.
- Fixed an issue related to the Roots update.
- Fixed an issue where water particles remained active after stopping pouring water.

## [1.0.0] - 2025-10-19

- Initial release.

# Watering Can

This mod adds the Watering Can, an item with multiple uses. Currently, eight uses have been implemented.

# ALL PLAYERS NEED THE MOD FOR IT TO WORK PROPERLY!

<details>
 <summary>Click to show current and upcoming mechanics</summary>

- Use it on bushes to grow Crispberries. (Implemented)
- Use it on palm trees to grow Coconuts. (Implemented)
- Use it on willows to grow Kingberries. (Implemented)
- Use it on vines to grow Berrynanas. (Not implemented)
- Use it on dead trees to grow Winterberries. (Implemented)
- Use it on cacti to grow Prickleberries. (Implemented)
- Use it on Magic Bean vine to grow it longer. (Not implemented)
- Use it on campfires to put them out. (Implemented)
- Use it on players to remove Burn units faster. (Implemented)
- Use it on lit dynamite to extinguish it. (Implemented / WARNING: Dynamites will relight upon grabbing. I plan to make them usable by allowing the player to light them manually, rather than automatically lighting themselves)
- Refill it at sea. (Implemented)
- Refill it slowly while it's raining. (Implemented)
- Refill it under waterfalls. (Implemented)
- Refill it in clean shallow water in Roots. (Implemented)
- Refill it in water pools. (Implemented)

</details>
